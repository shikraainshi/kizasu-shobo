# 萌書房 | Kizasu Shobo

奈良の地から、新世紀の思想を世に問う出版社、萌書房の公式ウェブサイトです。

## プロジェクト概要

- **ドメイン**: kizasu-shobo.jp
- **メールアドレス**: kizasu-s@m3.kcn.ne.jp
- **技術スタック**: Next.js (App Router), TypeScript, TailwindCSS, Framer Motion

## 開発ガイドライン

### 開発サーバーの起動

```bash
npm run dev
```

[http://localhost:3000](http://localhost:3000) でプレビュー可能です。

### デプロイ

Vercelなどのプラットフォームへのデプロイを想定しています。
